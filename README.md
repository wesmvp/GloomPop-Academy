# GloomPop Academy

GloomPop Academy is a browser-based profile collection prototype built with HTML, CSS, and JavaScript.

The project is currently hosted with GitHub Pages:

**Live site:** 
https://wesmvp.github.io/GloomPop-Academy/

## Overview

The current prototype focuses on a student ID/profile system. Players create a profile, receive a faction through a one-time quiz, collect items through daily rolls, and customize their ID with equipped items.

All data is currently stored locally in the browser using `localStorage`.

## Current Features

- Single-page website with no page reloads
- Student ID/profile card
- One-time faction quiz
- Randomized monster type selection
- Daily roll system with a 24-hour refresh
- Item categories:
  - Curses
  - Pets
  - ID colors
- Rarity system:
  - Common
  - Uncommon
  - Rare
  - Epic
  - Legendary
  - Forbidden
- Duplicate roll handling
- Customization page for equipped items
- Collection page with category tabs
- Local popularity score calculation
- Prototype rankings page
