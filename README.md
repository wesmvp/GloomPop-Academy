# GloomPop Academy

GloomPop Academy is a browser-based profile collection prototype built with HTML, CSS, JavaScript, GitHub Pages, Firebase Authentication, and Firestore.

## Live Site

https://r9kzbqrkqg.github.io/GloomPop-Academy/

## Overview

The prototype focuses on a student ID/profile system. Users can create an account, generate a student profile, take a one-time faction quiz, collect items through daily rolls, customize their ID, and build a popularity score.

Visitors can view the site and rankings. Account features require sign-in.

## Files

- `index.html`
- `style.css`
- `script.js`
- `firebase.js`
- `README.md`

## Notes

This is an early prototype intended to test the profile loop, account flow, collection system, and visual direction before adding public profiles, trading, events, and global chat.
